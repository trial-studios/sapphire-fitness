# Sapphire Fitness 💎

A futuristic premium fitness website built using Next.js, React, TailwindCSS, and Framer Motion.

## Features

- Responsive UI
- Animated sidebar
- Language switching
- Interactive FAQ
- Glitch 404 page
- Mouse glow effects
- Modern animations

## Tech Stack

- Next.js
- React
- TailwindCSS
- Framer Motion
